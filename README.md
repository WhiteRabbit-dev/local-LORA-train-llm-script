# local-LORA-train-llm-script
Данный скрипт позволит вам обучить/дообучить LLM локально на вашем пк совершенно бесплатно. 
Плюсы скрипта: 
- он загружает все библиотеки в свою папку не засерая пк не нужными библиотеками и файлами, 
- способен полностью обучить и подготовить к использованию llm абсолютно локально, 
- открытый код на python позволит вам что редактировать самостоятельно.
там еще есть но вспоминать лень))
К сожалению гилаб не позволяет загружать файлы более 25мб а мой скрипт использует тяжелые библиотеки которые вам к сожалению нужно будет докачать отдельно с моего гугл диска.
Все что вам нужно знать для использования указано в инструкции к программе: файл: "Инструкция.docx"
